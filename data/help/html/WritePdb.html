NAME

WritePdb - write PDB file
SYNOPSIS

WritePdb fileName
DESCRIPTION

Write a PDB file. All selected molecules are written to the file.

See the help page of ReadPdb for an explanation of the handling of atom names.

EXAMPLE

WritePdb /home/joe/struct/my_prot.pdb
